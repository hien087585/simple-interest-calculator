# Simple Interest Calculator

This project is a simple interest calculator implemented using a Bash script.

## Description
The script calculates simple interest based on user input:
- Principal amount
- Rate of interest
- Time period

## Usage
Run the script using:
bash simple-interest.sh

Then enter the required values when prompted.

## Example
Input:
Principal = 1000  
Rate = 5  
Time = 2  

Output:
Simple Interest = 100
